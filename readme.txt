Collective Art is a Plymouth University Final Year Project, created to explore interaction between users in a digital environment. It is also a project dedicated to researching new digital technologies and learning new skills that can be applied to digital applications.

The frontend application is designed using React, a Javascript UI library developed by Facebook, it is used in combination with React-Three-Fiber. RTF is a library that allows React to use Three js a WebGL library for the creation of 3D environments.
